# SkinCare Website

A single-page website for skincare products, built with HTML, CSS, and JavaScript. The site features long scrolling sections, well-designed product cards, animations, and a fully responsive layout. Developed based on a Figma mockup with personal improvements and enhancements.

# Features

Single-page layout: Smooth scrolling with clearly structured sections

Product cards: Detailed cards with images, descriptions, and hover animations

Animations: Interactive effects on hover and scroll

Responsive design: Works well on desktop, tablet, and mobile

Customization: Built with personal design improvements over the original mockup

# Tech Stack
- HTML
- CSS
- JavaScript (ES6+)

Gulp (task runner for build and automation)

# Installation
npm install

# Development server
gulp serve

# Production build
gulp build
